# Update Code

Time for the [*coup de grâce*](https://en.wikipedia.org/wiki/Coup_de_gr%C3%A2ce)!

Let's test our agent's ability to actually *fix a bug* all on its own.

## Assignment

1. [x] Manually update `calculator/pkg/calculator.py` and change the `precedence` of the `+` operator to `3`.
2. [x] Run the calculator app, to make sure it's now producing incorrect results: `uv run calculator/main.py "3 + 7 * 2"` (this should be `17`, but because we broke it, it will return `20`).
3. [x] Run your agent and ask it, "Fix the bug: `3 + 7 * 2` shouldn't be `20`."
